Part1:
后台->服务器数据格式定义：
register:
{
	state:0,
	name:<nickName>(string),
	phone:<phoneNumber>(string),
	psw:<password>(string)
}
login：
{
	state:1,
	acc:<account>(string),
	psw:<password>(string)
}

获取离线消息：
{
	state:2,
	myName:<userNickName>(string),
	friendName:<friendNameOnMyList>(string)
}

发送消息：
{
	state:3,
	targetName:<the message's destination>(string),
	message:<message contain>(string),
	name:<sender's nickName>(string)
}

查找好友：
{
	state:4,
	search:<phoneNumber>(string),
	requirer:<user who send this require>
}


Part2:

服务器->后台数据格式：
注：数据都用对象存储，发送前用JSON.stringify转换

login：
登录成功返回"success"

register:
若成功则返回"success"；
若有其它状态，佳斌可以在这里更新接口

离线消息发送：
{
    state:1,
 	log:[<object array>],
 	from:<who send this message>
 	/*
 	 object format:
 	 {
 	 	text:<message contain>,
 	 	from:"recv"
 	 }
 	*/
}

离线消息发送完毕：
{
	state:0
}

消息转发:
{
	state:3,
	text:<message from friend>(string),
	from:<friendName>(string)	
}

查找好友信息返回：
{
	state:<4 for has result and others for None>
	info:<the info of whose phoneNumber matches the prefix of the require>(object array)
	/*
		object:{
			name:(string),
			phoneNumber:(string),
			asume:<the asume which was filled when register>(string)
		}

	}
	 */
}
