Part1:
后台->服务器数据格式定义：
login：
{
    state:1,
    code:res.code(string)
}
获取离线消息：
{
  state:2,
  trd:3rd_session(string),
  seq:<confirm number>(int)
}

发送消息：
{
	state:3,
	friendSecret:<the message's destination>(string),
	message:<message contain>(string),
	trd:<sender's nickName>(string),
    seq:<confirm number>(int)
}

给服务器发送临时id，用于邀请好友进行通信：
{
	state:4,
	trd:3rd_session,
	tempId:临时ID
}


Part2:

服务器->后台数据格式：
注：数据都用对象存储，发送前用JSON.stringify转换

login：
登录成功返回:
{
    reply:<3rd>(string),
    pubkey:<server public rsa key>(string)
}
离线消息发送：
{
    state:1,
    
    seq:<client seq + 1>(int),
}

离线消息发送完毕：
{
	state:0,
    seq:<client seq + 1>(int)
}

消息转发:
{
	state:3,
	text:<message from friend>(string),
	from:<friendName>(string)	
}

查找好友信息返回：
{
	state:<4 for has result and others for None>
	info:<the info of whose phoneNumber matches the prefix of the require>(object array)
	/*
		object:{
			name:(string),
			phoneNumber:(string),
			asume:<the asume which was filled when register>(string)
		}

	}
	 */
}
